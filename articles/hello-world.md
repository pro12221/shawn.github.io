# 你好，世界！

这是我的第一篇 Markdown 文章，用来测试自动识别与渲染。

## 特性

- **自动识别**：把 `.md` 文件放进 `articles/` 目录即可出现在列表。
- **即时渲染**：点击列表中的文章标题即可看到渲染后的 HTML。
- **Mac 风格**：沿用网站的玻璃拟态与窗口设计。
